CSS属性display的inline、block、inline-block的区别

三种元素解释

block:：对象为块状元素，整体保持块状状态（矩形形状），并且单独占据一行（同一行内不会有兄弟节点）

inline：对象为行内元素，整体不会一定保持块状状态，并且不会一定单独占据一行，超过一行的部分进入下一行

inline-block：介于block与inline之间，整体保持块状状态，但不会一定单独占据一行
